Who I Am

A Software Engineer based off the Bay Area, CA

Graduated from University of California, Los Angeles with a B.S. in Computer Science. Through my education, I learned the fundamentals of Computer Science, such as data structures, algorithms, and networks. Through my personal projects I learned the practical application of that knowledge as well as the skills and tools required to succeed in software engineering role in the tech industry

As a first-generation immigrant as well as a first-generation university graduate, I have the drive and creativity to create opportunities for myself

My experience in backend development includes writing in Node.js, JavaScript, and TypeScript. I have experience in building both a REST API and a GraphQL API. I also have experience working with noSQL databases such as MongoDB, and relational databases such as PostgreSQL and MySQL. I have also worked with services such as AWS and Twilio SendGrid.

My experience with all of these different technologies allows me to determine the best solution to a problem, and I have the ability to refactor the codebase from using one technology to another. I have refactored a REST API into a GraphQL API. As well as changing the database from using MongoDB to using PostgreSQL

My skills in frontend development include React, JavaScript, TypeScript, and HTML & CSS. I also have experience with component libraries such as MUI. With my intuition for good UI/UX design, I'm able to create functional and responsive web applications that looks great on any screen size.
